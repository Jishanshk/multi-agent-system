# 🤖 Multi-Agent AI System

An Autonomous Multi-Agent AI application built using **Python, LangChain, and Streamlit**. The system understands user requests, creates an execution plan, performs tasks using AI agents, and generates structured Word reports automatically.

## 🚀 Features

- Multi-Agent Architecture
- Task Planning & Execution
- AI-powered Workflow
- Streamlit User Interface
- Automated Report Generation (.docx)
- Environment Variable Support (.env)
- Modular Python Code

## 🛠️ Tech Stack

- Python
- LangChain
- Streamlit
- python-dotenv
- python-docx
- Rich

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/multi-agent-system.git
```

Move into the project

```bash
cd multi-agent-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your API key.

Example:

```env
GROQ_API_KEY=your_api_key
```

Run the application

```bash
streamlit run app.py
```
